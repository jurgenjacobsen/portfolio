# Live Wallpaper
Live Wallpaper is a desktop application that allows users to set dynamic wallpapers on their desktop. It provides a seamless experience for users to customize their desktop environment with dynamically updated wallpapers.

## What it does?
- Ships as a native desktop application for multiple platforms.
- Runs in the background and updates the wallpaper based on user preferences.
- Per-monitor wallpaper customization, allowing different wallpapers for each connected display.
- Scheduled refresh with configuration intervals.
- Tray controls for quick access to settings and wallpaper management.
- Automatic update check on startup via GitHub releases.

## Supported "Widgets"
- **Weather**: Displays current/forecast weather information and updates the wallpaper accordingly.
- **Aviation Weather**: Provides aviation-specific weather information for pilots and aviation enthusiasts. (METAR/TAF)
- **Plane.so Board View**: Displays a live board view of work items from a Plane.so workspace, allowing users to track their project items in the desktop environment.

## Technical Details
- **Backend**: Go
- **Frontend**: TypeScript, React
- **Size**: < 20MB
- **Memory Usage (RAM)**: < 100MB
- **Cross-Platform**: Supports Windows, macOS, and Linux.
